BootStrap
    Bootstrap 是一个用于快速开发 Web 应用程序和网站的前端框架。
    Bootstrap 是基于 HTML、CSS、JAVASCRIPT 的。

    特点:
        Responsive web page 响应式/自适应的网页
        可以根据浏览器的设备不同(PC,PAF,PHONE)，自动调整对应的布局

响应式布局
    1. <meta name="viewport" content="width=device-width,initial-scale=1">  响应式布局标签 
        viewport 视口   : 呈现在用户窗口的内容，超出视口的内容需要滑动查看           
        content  设置能够容许网页进行的操作
        width=device-width  表示视口宽度就是设备宽度
        initial-scale=1   视口宽度是否可以缩放 1能缩放  1.0不能缩放
        maximum-scale=1.0   允许缩放的最大倍率 
        user-scalable=0     是否允许用户缩放 1允许 0不允许   

    2. 所有内容要使用相对尺寸，不能使用绝对尺寸
    3. 流式布局+弹性布局，在搭配媒体查询技术构建响应式
