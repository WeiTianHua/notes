
pip 的使用

作用 : 用于安装python的标准第三方库

安装 : sudo apt-get install python3-pip

常用命令 :
    安装包 :sudo pip3 install [package]

    查看安装的python包 : pip3 list

    搜索python包 : pip3 search [package]

    升级软件包 : pip3 install --upgrade [package]

    查看软件包的信息 : pip3 show [package]

    卸载软件包  :  pip3 uninstall [package]

    导出软件包环境  :  pip3 freeze > [requirements.txt]

    安装软件环境 : pip3 install -r [requirements.txt] #在目标主机
